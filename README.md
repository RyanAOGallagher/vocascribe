## Gemini API Setup

To use the transcription analysis feature, you need to set up a Gemini API key:

1. Go to [Google AI Studio](https://makersuite.google.com/app/apikey) to get your API key
2. Create a `.env.local` file in the root directory
3. Add your API key: `GEMINI_API_KEY=your_actual_api_key_here`
4. Restart your development server

The Extract button will now send transcriptions to Gemini for analysis of vocabulary, grammar patterns, and insights.
